About

Typy is an experimental TypeScript MVVM framework.

Feel free to submit any observation or issue.

Typy is not intended to be used in production, only for educational purposes.

Github address: https://github.com/rzvpopescu/typy

Example

Summary:

                
                I'm ${person.name} . I'm a ${person.age} years old ${person.gender}. 
                Currently I'm .  My current workplace is ${person.employer}

1. Go to app folder and create a pair of files(MyApp.html and MyApp.ts).

MyApp.ts : 

                   export default class MyApp{
                        title:string = 'Hello Guest';
                        
                        person:any = {
                            name:'John',
                            age:22,
                            gender:'male',
                            employeed:true,
                            employer:'BigCorp Company'
                        }

                        genders:Array = ['male','female'];

                        displaySummary(){
                            let message = `I'm ${this.person.name}. I am a ${this.person.age} years old ${this.person.gender} !`;
                            alert(message);
                        }
                    
                    }
MyApp.html : 
                    <div>
                        <h4> ${title}</h4>
                        <div class="form-group">
                            <label for="name" class="col-sm-2 control-label">Name</label>
                            <div class="col-sm-10">
                                <input type="text" class="form-control" value.bind="person.name" id="name" placeholder="name">
                            </div>
                        </div>
                        <div class="form-group">
                            <label for="age" class="col-sm-2 control-label">Age</label>
                            <div class="col-sm-10">
                                <input type="number" class="form-control" value.bind="person.age" id="age" placeholder="20">
                            </div>
                        </div>
                        
                        <div class="form-group">
                            <label for="gender" class="col-sm-2 control-label">Gender</label>
                            <div class="col-sm-10">
                                <select class="form-control" id="gender" value.bind="person.gender" for.bind="gender of genders">
                                    <option value.bind="gender">${gender}</option>                                        
                                </select>
                            </div>
                        </div>
                        <div class="form-group">
                            <label for="empl" class="col-sm-2 control-label">Employeed</label>
                            <div class="col-sm-10">
                            <div class="checkbox">
                                <label><input type="checkbox" checked.bind="person.employeed" ><span text.bind="person.employeed ? '(Employeed)':'(Unemployeed)'"></span></label>
                                </div>
                            </div>
                        </div>
                        <div class="form-group" visible.bind="person.employeed">
                            <label for="emply" class="col-sm-2 control-label">Employer</label>
                            <div class="col-sm-10">
                                <input type="text" class="form-control" value.bind="person.employer" id="emply" placeholder="employer">
                            </div>
                        </div>
                        
                        <div class="form-group">
                            <button class="btn" click.bind="displaySummary()">Display summary</button>
                        </div>
                    </div>

2. Go to index.html and select the created TypeScript file as the start-up page (MyApp.ts). 

                        <!DOCTYPE html>
                            <html lang="en">
                            <head>
                            <meta charset="UTF-8">
                            <title>Welcome to Typy !</title>
                            <link rel="stylesheet" href="jspm_packages/github/twbs/bootstrap@3.3.6/css/bootstrap.min.css">  
                            <script src="jspm_packages/system.js"></script>
                            <script src="config.js"></script>
                            <script>                                    
                                System.import('lib/Application.ts').then(function(app){      
                                app.Application.startUp('../app/MyApp')
                                });
                            </script>
                            </head>
                            <body>
                            <div id="main-host" class="container">     
                            </div>
                            </body>
                            </html>
